# PMHC.co portfolio
This can serve as a React boilerplate for future projects.

##Features
- Universal React
- redux/react-redux
- react-router
- redux-connect (AsyncPropsConnect)
- Development
  - WebpackDevServer
  - Hot module replacement
  - ReduxDevTools
- Webpack-isomorphic-tools (coming soon)

Check out webpack config for the loaders available.
